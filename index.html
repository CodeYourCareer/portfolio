Project: Secondary Radar Monopulse Tuner — Key Technical Steps

01
Data Extraction
Parsed UDP packets from pcap files → extracted code, sigma, delta, ACP fields → saved as CSV.
02
ACP Filtering
Applied beam-width criteria, removed outliers, validated positive/negative boresight side classification.
03
Target Selection
Selected Top-10 most consistent scan targets; applied 100 km range filter to remove optical interference noise.
04
Obi Calculation
Computed sigma–delta (obi) values for each ACP across all 10 targets within 100 km range.
05
Equation Fitting
Fitted cubic polynomial y = ax³+bx²+cx+d to obi vs ACP data for monopulse equation derivation.
06
GUI Output
Generated calibration_load.csv for GUI display; visualised monopulse curve with Matplotlib.
